fish
=========

This role will setup fish.

Role Variables
--------------

fish_version - This version will be installed (default:3.4.1-1).
fish_config_commands - command list which you want to add to config.fish (default: []).
fish_external_environment_paths - PATH list which you want to add (default: []).
